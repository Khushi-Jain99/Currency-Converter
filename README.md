# Currency Converter

A web-based application that allows users to convert between different global currencies using real-time exchange rates. The project focuses on providing a clean, minimal, and intuitive user experience.

## Features

* Convert between multiple currencies
* Simple and minimal UI
* Fast and responsive design

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

* Clone the repository
* Open the project folder
* Run using Live Server or open index.html in a browser
